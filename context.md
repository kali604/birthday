## Technical Decisions & Preferences

- **No frameworks**: Pure vanilla HTML/CSS/JS. No build step. No dependencies.
- **Data-driven**: All content lives in JSON files. The engine is generic.
- **Debug mode**: `?debug` URL param for fast testing
- **Cache busting**: `?v=N` on script tag (currently v=5)
- **Server**: `python3 -m http.server 8080`
- **Font stack**: Playfair Display (headings) + Lora (body) from Google Fonts
- **Animations**: CSS keyframes (float, fadeInUp, riseUp, confettiFall). JS creates/removes DOM elements.
- **Music**: HTML5 `<audio>` with loop. Two tracks that switch based on scene. Toggle button top-right.
- **The user edits story.json directly**: Always check current file contents before editing. Never overwrite without reading first.
